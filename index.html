These are graphs of results from the enmeaten Node module. It's a module that figures out how to widen an elbow point, given two lines that form an elbow.

45 degree angle

Input: 

  {
    start: [30, 50],
    elbow: [0, 20],
    end: [20, 0],
    widenDistance: 20
  }
Result: 

  [
    [-20, 20],
    [20, 20]
  ]

135 degree angle

Input: 

  {
    start: [0, 100],
    elbow: [-50, 50],
    end: [-50, 0],
    widenDistance: 130
  }
Result: 

  [
   [-170.104, 99.748],
   [70.104, 0.251] 
  ]
